# Background
This CTF has 15 categories:
1. Warmups
2. Web
3. Binary Exploitation
4. Reverse Engineering
5. Cryptography
6. Forensics
7. Mobile
8. DevOps
9. Hardware/RF
10. Malware
11. Miscellaneous
12. Networking
13. Scripting
14. Steganography
15. OSINT

# Rules
1. No automated scanners like sqlmap, DirBuster, nmap, Metasploit, nikto or others.
